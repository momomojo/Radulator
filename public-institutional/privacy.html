Radulator Institutional

Privacy

This institutional static build is designed to run in the browser without transmitting calculator inputs, calculator results, search terms, selected calculators, copied results, downloads, or feedback payloads.

  • No telemetry or advertising is included in the institutional artifact.
  • No feedback form, name field, email field, or message field is included.
  • No EHR, PACS, Epic, API, PHI, or institutional-network integration is implemented.
  • Emergency release control is a first-party static JSON file.

This page describes the implemented static build controls. Any institution-specific privacy terms, deployment controls, or support channels require owner and counsel review before release.
