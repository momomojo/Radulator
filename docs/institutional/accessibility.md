# Radulator Institutional Accessibility Statement

Status: implementation draft for release review.

Radulator Institutional uses the same React and Tailwind UI foundation as the public site and is checked for:

- semantic headings and landmarks;
- skip link to main content;
- keyboard-accessible controls;
- explicit button types;
- labeled form controls when forms exist;
- visible focus states;
- responsive layout;
- color contrast consistent with the existing design tokens.

This empty-catalog release contains no calculator input controls. Future non-empty releases must rerun accessibility checks against each included calculator.

Accessibility support-channel language is deployment-specific and requires owner approval before release.
