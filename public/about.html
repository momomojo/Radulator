About Radulator

Mission: Provide free, accurate, evidence-based medical calculators for healthcare professionals, trainees, and researchers.

What is Radulator?

Radulator is a collection of medical calculators focused on radiology, hepatology, urology, and interventional medicine. Each calculator is built using formulas from peer-reviewed literature, with references to the original studies.

All calculations are performed locally in your browser. We never collect, store, or transmit any patient data or clinical information you enter.

Methodology

Each calculator includes:

  • Source References: Links to the original peer-reviewed studies
  • Validated Formulas: Calculations verified against published literature
  • Clinical Context: Information about appropriate use and limitations

Technology

Radulator is built with modern web technologies:

  • React 19 for the user interface
  • Tailwind CSS for responsive design
  • Vite for fast development and optimized builds
  • Playwright for comprehensive testing

The entire application runs client-side, ensuring your data privacy and enabling offline use once loaded.
